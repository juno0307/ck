家庭财务金融计算器

复利现值/终值计算

请输入任意三个数值，留空一个待计算的值，然后点击"计算"按钮。

📐 涉及公式
FV = PV × (1 + i)^n PV = FV / (1 + i)^n i = (FV / PV)^(1/n) - 1 n = ln(FV / PV) / ln(1 + i)

年金现值/终值计算

选择年金类型，输入已知参数，点击对应按钮计算未知项。
注意：选择“递延年金”时，需输入递延期数。

📐 涉及公式
普通年金现值: PV = PMT × [1 - (1+i)^-n] / i 普通年金终值: FV = PMT × [(1+i)^n - 1] / i 预付年金: 上述结果 × (1+i) 递延年金现值: PV = (普通年金现值) / (1+i)^m 或 PV = PMT × [ (P/A, i, n+m) - (P/A, i, m) ] 其中： PMT = 每期年金金额 m = 递延期数 (仅递延年金有效) i = 每期利率 n = 年金发生期数 注：利率i和期数n的精确计算通常需迭代法求解。

名义利率与实际利率转换

基于费雪方程：1 + r = (1 + i) / (1 + π)，输入任意两个值计算第三个。

📐 涉及公式
精确费雪方程: 1 + r = (1 + i) / (1 + π) 推导公式: 实际利率: r = [(1 + i) / (1 + π)] - 1 名义利率: i = (1 + r) × (1 + π) - 1 通胀率: π = (1 + i) / (1 + r) - 1

项目风险衡量

输入不同市场情景的发生概率及各项目收益率，自动计算预期收益、方差、标准差及标准离差率。

📐 涉及公式
1. 预期收益率 E(R): E(R) = Σ [ P(i) × R(i) ] 2. 方差 σ²: σ² = Σ [ P(i) × (R(i) - E(R))² ] 3. 标准差 σ: σ = √σ² 4. 标准离差率 CV: CV = σ / |E(R)|
